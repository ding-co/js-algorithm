## 4. 1부터 N까지 합 출력하기

### _문제 풀이_

```html
<!-- My Solution -->

<html>
  <head>
    <meta charset="UTF-8" />
    <title>1부터 N까지 합</title>
  </head>
  <body>
    <script>
      function solution(n) {
        let answer = 0;
        for (let i = 0; i <= n; i++) {
          answer += i;
        }
        return answer;
      }

      console.log(solution(10));
    </script>
  </body>
</html>
```
