## 7. 10부제

### _문제 풀이_

- % 10 => 항상 1의 자리 숫자 확인 가능

```html
<!-- My Solution -->

<html>
  <head>
    <meta charset="UTF-8" />
    <title>10부제</title>
  </head>
  <body>
    <script>
      function solution(day, arr) {
        let answer = 0;
        for (let i = 0; i < arr.length; i++) {
          const carNumber = arr[i] % 10;
          if (day === carNumber) {
            answer += 1;
          }
        }
        return answer;
      }

      arr = [12, 20, 54, 30, 87, 91, 30];
      console.log(solution(0, arr));
    </script>
  </body>
</html>
```
