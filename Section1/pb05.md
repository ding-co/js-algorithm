## 5. 최솟값 구하기

### _문제 풀이_

```html
<!-- My Solution -->

<html>
  <head>
    <meta charset="UTF-8" />
    <title>최솟값</title>
  </head>
  <body>
    <script>
      function solution(arr) {
        let answer;
        let minValue = arr[0];
        for (let i = 1; i < arr.length; i++) {
          if (minValue > arr[i]) {
            minValue = arr[i];
          }
        }
        answer = minValue;
        return answer;
      }
      let arr = [5, 7, 1, 3, 2, 9, 11];
      console.log(solution(arr));
    </script>
  </body>
</html>
```

```html
<!-- Lecture Solution  -->

<html>
  <head>
    <meta charset="UTF-8" />
    <title>최솟값</title>
  </head>
  <body>
    <script>
      function solution(arr) {
        let answer;

        // 미리 안정적인 큰 숫자로 초기화
        // min 값을 arr[0]로 초기화 해서 풀어도 상관없음
        let min = Number.MAX_SAFE_INTEGER;

        for (let i = 0; i < arr.length; i++) {
          if (arr[i] < min) min = arr[i];
        }
        answer = min;
        return answer;
      }
      let arr = [5, 7, 1, 3, 2, 9, 11];
      console.log(solution(arr));
    </script>
  </body>
</html>
```

### [Note]

- 본인 스스로 테스트 데이터를 만들어서 검증해보는 훈련 필요!
- 참고 풀이 (reduce 메서드 이용)

  ```js
  <html>
  <head>
    <meta charset="UTF-8" />
    <title>최솟값</title>
  </head>
  <body>
    <script>
      function solution(arr) {
        // return arr.reduce((acc, cur) => Math.min(acc, cur));
        return arr.reduce(
          (acc, cur) => Math.min(acc, cur),
          Number.MAX_SAFE_INTEGER
        );
      }
      let arr = [5, 7, 1, 3, 2, 9, 11];
      console.log(solution(arr));
    </script>
  </body>
  </html>

  ```
